# ­ЪЈд BOG Donation Plugin for WordPress

**BOG Donation** рЃљрЃарЃўрЃА WordPress-рЃўрЃА рЃърЃџрЃљрЃњрЃўрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮрЃЌ **рЃЮрЃюрЃџрЃљрЃўрЃю рЃЊрЃЮрЃюрЃљрЃфрЃўрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ** рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃЉрЃљрЃюрЃЎрЃўрЃА (BOG) рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃАрЃЌрЃљрЃю рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃЌ.

## Рюе рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

- ­Ъњ│ рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃЉрЃљрЃюрЃЎрЃўрЃА рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃћрЃЉрЃўрЃА API рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ
- ­Ъћљ OAuth2 рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў
- ­ЪњИ рЃЊрЃЮрЃюрЃљрЃфрЃўрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЊрЃљ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃЌрЃљрЃюрЃ«рЃўрЃЌ
- ­ЪДЙ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ WordPress рЃЉрЃљрЃќрЃљрЃерЃў
- ­ЪЊЕ рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃўрЃЌ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџ рЃЊрЃЮрЃюрЃљрЃфрЃўрЃћрЃЉрЃќрЃћ
- ­ЪДа рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ (.po/.mo рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃЌ)
- ­ЪћД рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃўрЃЊрЃљрЃю рЃДрЃЋрЃћрЃџрЃљ API рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
- ­ЪЊі рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ
- ­ЪДЕ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃерЃЮрЃарЃбрЃЎрЃЮрЃЊрЃў: `[bog_donation_form]`

## ­ЪЏа№ИЈ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

1. рЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃърЃџрЃљрЃњрЃўрЃюрЃў WordPress admin рЃърЃљрЃюрЃћрЃџрЃўрЃЊрЃљрЃю (`Plugins Рєњ Add New Рєњ Upload Plugin`)
2. рЃњрЃљрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЌ
3. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `BOG Donation Рєњ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў` рЃЊрЃљ рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ:
   - `Client ID`
   - `Client Secret`
   - `Callback URL` рЃЊрЃљ рЃАрЃ«рЃЋрЃљ
4. рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃерЃЮрЃарЃбрЃЎрЃЮрЃЊрЃў `[bog_donation_form]` рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
5. рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ redirect URL (рЃЏрЃљрЃњ., /thanks) рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЊрЃљрЃАрЃљрЃарЃЌрЃљрЃЊ

## ­ЪЊг рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃўрЃЌ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ

рЃДрЃЋрЃћрЃџрЃљ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃљрЃќрЃћ (`admin_email`).

## ­Ъїљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ

рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃњрЃљрЃюрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЌ `languages/` рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ `load_plugin_textdomain()`.  
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃцрЃљрЃўрЃџрЃў: `bog-donation-ka_GE.po`

## ­ЪДф рЃАрЃљрЃбрЃћрЃАрЃбрЃЮ рЃњрЃљрЃарЃћрЃЏрЃЮ

рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ BOG-рЃўрЃА **sandbox API** рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.  
Рєњ [API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ](https://api.bog.ge/docs/payments/introduction)

## РюЁ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ

- рЃцрЃЮрЃарЃЏрЃљ рЃЊрЃљрЃфрЃБрЃџрЃўрЃљ `nonce` рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃЌ
- admin рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃерЃћрЃќрЃдрЃБрЃЊрЃБрЃџрЃўрЃљ `manage_options` рЃБрЃцрЃџрЃћрЃЉрЃљрЃќрЃћ
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ `sanitize_*` рЃЊрЃљ `esc_*` рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃЌ
- `wp_mail()` рЃЊрЃљ `wpdb` рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ WordPress-рЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ
